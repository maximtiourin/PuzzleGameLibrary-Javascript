(static) redirect(urlnon-null, openNewWindowopt, nullable, isLinkopt, nullable) → {Boolean}

Opens a link to the given url, either in the current window, or a new window/tab. The redirection can allow browser history to be written by simulating a user clicking a link, or prevent history from being written by using an http redirect.

Absolute redirect ex: "http://www.myurl.com/test.html"
Relative redirect in current directory ex: "/test2.html"
Relative redirect in directory one level above ex: "../test3.html"
Parameters:
Name Type Attributes Default Description
url String The absolute or relative url string to redirect to
openNewWindow Boolean <optional>
<nullable>
 false Whether or not to open a new window/tab when redirecting
isLink Boolean <optional>
<nullable>
 true A value that is true to simulate the user clicking on a link and storing browser history, or false to simulate an http redirect and store no browser history.
