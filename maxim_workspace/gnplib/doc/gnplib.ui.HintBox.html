HintBox

gnplib.ui. HintBox

HintBox

Constructor

new HintBox(stagenon-null, maxWidthopt, nullable)

A smart HintBox class that displays a small frame with a given maxWidth and a dynamic height, that contains a createjs.Text object which wraps inside of it. The frame is closed by clicking inside of it.
Parameters:
Name Type Attributes Default Description
stage createjs.Stage The current createjs.Stage context to display the HintBox in
maxWidth Number <optional>
<nullable>
 200 The maximum pixel width of the hint box, to which the contents will constrain to
Properties:
Name Type Description
stage createjs.Stage [READ ONLY] The createjs.Stage context that this HintBox belongs to
box createjs.Container The createjs.Container object that holds all of the elements of the HintBox. Access this property to move the HintBox, scale it, etc.
maxWidth Number [READ ONLY] The maximum width of the HintBox.
maxHeight Number [READ ONLY] The maximum height of the HintBox.
width Number [READ ONLY] The width of the HintBox that is available to display content, calculated by accounting for padding of the left and right sides of the HintBox
height Number [READ ONLY] The height of the HintBox that is available to display content, calculated by accounting for padding of the top and bottom sides of the HintBox
text String [READ ONLY] The string value of the text to display inside of the HintBox
textColor Color The color to set the text of the HintBox to the next time setText() is called. Default value = "#000000"
textFont Font The font to use for the text of the HintBox the next time setText() is called. Default value = "12px Arial"
bgColor Color The color of the background for the HintBox. Must be set to render the HintBox. Default value = "#ffffff"
cornerRadius Number The corner radius of the background rectangle, values greater than 0 create rounded corners. Only used when there is a background color. Default value = 0
isDisplayed Boolean [READ ONLY] Is true if the HintBox is currently displayed, false if it is hidden

Example
//Creating and displaying a HintBox, centered on the point(400, 300)
var hintbox = new gnplib.ui.HintBox(myStage, 200); //Create a new HintBox attached to myStage, with a maxWidth of 200
hintbox.textColor = "#ff0000"; //Set text to red
hintbox.textFont = "16px Arial"; //Set font to size 16 Arial
hintbox.bgColor = "#0000ff"; //Set background color to blue
hintbox.cornerRadius = 15; //Set the HintBox to have rounded corners of radius 15
hintbox.setText("Hello! This is some text that will be contained within the HintBox..."); //Set the text of the HintBox and let it dynamically size itself
hintbox.centerOnPoint(400, 300); //Center HintBox on the point(400, 300)
hintbox.showDialog(true); //Display the HintBox
