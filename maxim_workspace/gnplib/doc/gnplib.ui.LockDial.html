gnplib.ui. LockDial

LockDial

Constructor

new LockDial(stagenon-null, bitmapImagenon-null, ticksInFullRotationnon-null, eventFuncopt, nullable)

A LockDial class that simulates a lock dial by allowing a user to rotate it by dragging. It then can return a value based on its rotation, taking into account how many ticks are supposed to be in a full rotation. A dial's value is 0 at the 90 degree position (top of dial), and increases in value in the clockwise direction, rolling over back to 0 when it reaches 90 degrees again.
Parameters:
Name Type Attributes Default Description
stage createjs.Stage The current createjs.Stage context to display the LockDial in
bitmapImage createjs.Bitmap The createjs.Bitmap object to display as the visual representation of the LockDial
ticksInFullRotation Number How many ticks are in a full 360 degree rotation of the dial
eventFunc NumberCallback <optional>
<nullable>
 null The callback function that will execute whenever the dial's value is updated, it will be passed the value of the dial as the first parameter
Properties:
Name Type Description
fullTicks Number [READ ONLY] How many ticks are contained inside of a full rotation of the LockDial
tick Number [READ ONLY] The current tick value of the LockDial, on the range [0, fullTicks)
lock createjs.Container The createjs.Container object that contains all of the visual elements of the LockDial. Use this to move, scale, etc the LockDial
