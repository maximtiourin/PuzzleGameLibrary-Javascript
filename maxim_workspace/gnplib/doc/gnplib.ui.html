(static) addDragAndDropToObject(displayObjectnon-null, startDragFuncopt, nullable, dragFuncopt, nullable, stopDragFuncopt, nullable, snappingObjectopt, nullable)

Applies drag and drop functionality to a createjs.DisplayObject, which allows the user to drag and drop the createjs.DisplayObject while optionally executing callback functions during the start of a drag, the continuation of a drag, and the end of a drag.
Parameters:
Name Type Attributes Description
displayObject createjs.DisplayObject The createjs.DisplayObject to apply the drag and drop functionality to
startDragFunc MouseCallback <optional>
<nullable>
 The function to execute when a drag is first initiated on the display object
dragFunc MouseCallback <optional>
<nullable>
 The function to execute when the display object is dragged, changing its position
stopDragFunc MouseCallback <optional>
<nullable>
 The function to execute when a dragged display object is dropped
snappingObject Object <optional>
<nullable>
 [Advanced use only] An object that has special x() and y() method functions that set the x and y position of the display object in a unique way, for example snapping to position. If this object is supplied, the drag and drop functionality will change how it assigns x and y positions to prevent breaking the special positioning. An example of a valid snappingObject is a gnplib.ui.PuzzlePiece. If a snappingObject is supplied, then a stopDragFunc should also be supplied and should return true after being called if that snappingObject has modified its x or y position on its own during the callback.
Author:
  • Maxim Tiourin
Source:
  • gnplib.js, line 221
Examples
//Adding simple drag and drop functionality to a createjs.DisplayObject
gnplib.ui.addDragAndDropToObject(myDisplayObject);
//Adding drag and drop functionality to a createjs.DisplayObject with callback functions
var startDrag = function(event) {
    //Do stuff when object has first initiated its drag
}

var doDrag = function(event) {
    //Do stuff when object has been dragged
}

var stopDrag = function(event) {
    //Do stuff when object has been dropped
}

gnplib.ui.addDragAndDropToObject(myDisplayObject, startDrag, doDrag, stopDrag);

Generates drag and droppable puzzle piece shapes from an image and returns a two dimensional array of the pieces, which are represented as gnplib.ui.PuzzlePiece wrapper objects.

Name Type Attributes Default Description

 The current createjs.Stage context to automatically add the puzzle pieces to as children. If stage is set to null, the pieces will need to be manually added as children to a createjs.Stage object.

height Number The total height of the puzzle, to which the heights of the individual puzzle pieces in a column will sum up to

columns Number How many columns of puzzle pieces to have in the puzzle

 0.40 A value on the range [0.00, 1.00] that determines the alpha value of the outline border of the individual puzzle pieces.
shouldSnapToPosition Boolean <optional>
<nullable>
 false Whether or not to allow puzzle pieces to snap into their original x and y position when dragged near it.
puzzleCompleteFunc EmptyCallback <optional>
<nullable>
 null A function to run when all of the puzzle pieces are snapped into position after a puzzle piece has been dragged. Requires snapping to be enabled.

Examples

             //Do this stuff when my puzzle is fully completed by a user!
